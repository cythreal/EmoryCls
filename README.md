# EmoryCls
LaTeX Class for Emory Dissertation

Package Requirements:
LaTeX2e
ifthen
fancyhdr
graphix
amsmath
amssymb
color
mhchem
titlesec
titletoc

Standard LaTeX distribution should include all required packages

Default BibTeX driver is biblatex+biber

Run:
pdflatex Sample.tex
biber Sample.bcf
pdflatex Sample.tex
to generate sample pdf
